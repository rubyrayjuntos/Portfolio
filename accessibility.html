Accessibility Statement

Last updated: January 2025

Our Commitment

Ray Swan is committed to ensuring digital accessibility for people with disabilities. I am continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

This website strives to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities and more user-friendly for everyone.

Accessibility Features

This website includes the following accessibility features:

  • Keyboard Navigation: All interactive elements can be accessed using only a keyboard
  • Screen Reader Support: Proper heading structure, alt text for images, and ARIA labels
  • High Contrast Mode: Dark theme option for better visibility
  • Focus Indicators: Clear visual focus indicators for keyboard navigation
  • Skip Links: Skip to main content links for faster navigation
  • Responsive Design: Works on all screen sizes and devices
  • Reduced Motion: Respects user preferences for reduced motion
  • Semantic HTML: Proper use of HTML elements for better structure

Known Limitations

While I strive to ensure the highest level of accessibility, some content may have limitations:

  • Some older portfolio images may lack detailed alt text descriptions
  • Complex animations may not be fully accessible to all users
  • Third-party content (if any) may not meet the same accessibility standards

Browser Compatibility

This website is designed to work with the following browsers and assistive technologies:

  • Chrome, Firefox, Safari, and Edge (latest versions)
  • Screen readers: NVDA, JAWS, VoiceOver, and TalkBack
  • Voice recognition software
  • Keyboard-only navigation
